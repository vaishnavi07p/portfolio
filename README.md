## Run in VS Code

1. Extract the ZIP file.
2. Open VS Code.
3. Select `File > Open Folder`.
4. Open the extracted portfolio folder.
5. Install the `Live Server` extension.
6. Right-click `index.html`.
7. Select `Open with Live Server`.

## Upload to GitHub

1. Open the existing repository `vaishnavi07p/portfolio`.
2. Click `Add file`.
3. Select `Upload files`.
4. Upload all extracted files and the `assets` folder.
5. Click `Commit changes`.

## Enable GitHub Pages

1. Open repository `Settings`.
2. Select `Pages`.
3. Choose `Deploy from a branch`.
4. Select branch `main`.
5. Select folder `/ (root)`.
6. Click `Save`.

`https://vaishnavi07p.github.io/portfolio/`
